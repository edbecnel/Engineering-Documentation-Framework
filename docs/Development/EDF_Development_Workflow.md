> **Owner:** Engineering Documentation Framework

> **Applies To:** EDF repository only

## Workflow

1. Update architecture.
2. Update documentation.
3. Update templates.
4. Update generators.
5. Update Framework Advisor.
6. Update governance when applicable.
7. Implement.
8. Validate.
9. Perform self-host validation.
10. Update `CHANGELOG.md`.
11. Release.
